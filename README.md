# verilog-synchronous-fifo
Parameterizable synchronous FIFO implemented in Verilog with testbench and simulation.

# Synchronous FIFO using Verilog

This is one of my Verilog projects that I built while learning RTL design. The main aim was to understand how a synchronous FIFO works internally, especially how read/write pointers, memory, and full/empty conditions are handled.

While working on this project, I learned a lot about pointer wrap-around, parameterized design, writing testbenches, and debugging simulation results. There were quite a few bugs along the way, but fixing them helped me understand FIFO operation much better.

### What's included

* FIFO design (`FIFO_buffer.v`)
* Testbench (`FIFO_tb.v`)
* Simulation waveform

### Tools

* Verilog HDL
* Visual Studio Code
* Icarus Verilog
* GTKWave
